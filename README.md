# Polling Daly BMS Modbus Register Addresses

## Technology Stack
- Python >3.9
- PySerial


## clone repository
```bash
    git clone https://github.com/your-username/python-daly-16S.git
    cd python-daly-16S
```

## install dependencies
```bash
    pip install -r requirements.txt
```

## Klik kanan pada file `new_main.py` lalu pilih `Open in Integrated Terminal`

## jalankan perintah ini di terminal
```bash
    python new_main.py
```
